🗺️ ゴールデンファイル作成用 Google Maps リンク集

📋 使用方法

  1. 各ルートの「9時出発」または「10時到着」リンクをクリック
  2. Google Mapsが開いたら、時刻設定を確認してください
    • もし現在時刻になっている場合は、「出発時刻」をクリックして手動で設定
    • 9時出発: 平日の朝9:00を設定
    • 10時到着: 平日の朝10:00に到着を設定
  3. 表示されたルート情報（総時間、徒歩時間、駅名、路線名など）をメモ
  4. 下記のデータテンプレートを参考に、JSONファイルに記入
  5. verified_manually: true に変更するのを忘れずに！

📝 データ記入テンプレート

{ "total_time": [総所要時間（分）], "walk_to_station": [出発地から駅までの徒歩時間（分）], "station_used": "[最初に使用する駅名（「駅」を除く）]", "trains": [ { "line": "[路線名（例：地下鉄銀座線）]", "time": [乗車時間（分）], "from": "[出発駅名（「駅」を除く）]", "to": "[到着駅名（「駅」を除く）]" } ], "walk_from_station": [最後の駅から目的地までの徒歩時間（分）] }

💡 記入時の注意点

  • 駅名から「駅」を除去する（例：「神田駅」→「神田」）
  • 路線名は正確に記入（例：「地下鉄銀座線」「JR山手線」）
  • 時間は分単位で記入（秒は切り捨て）
  • 複数の電車を使う場合は、trains配列に順番に追加
  • 待ち時間（wait_time_minutes）は常に3分で固定
