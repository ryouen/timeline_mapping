5. Possible Solutions

  1. Apache: mod_php must be installed and enabled
  2. Nginx + PHP-FPM: PHP-FPM must be running
  3. .htaccess: May need AddHandler or SetHandler directive
  4. File permissions: PHP files need correct ownership

6. Quick Fix Attempts

Try these .htaccess configurations:

# Option 1: Basic PHP handler
AddHandler application/x-httpd-php .php

# Option 2: Specific handler
<FilesMatch "\.php$">
    SetHandler application/x-httpd-php
</FilesMatch>

# Option 3: Alternative handler
AddType application/x-httpd-php .php
